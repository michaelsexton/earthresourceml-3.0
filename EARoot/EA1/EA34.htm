-- Definition --<br/>A spatial representation of a MiningFeature<br/><br/>-- Description --<br/>A MiningFeatureOccurrence provides a link between a notional feature (description package) and one spatial representation of it, or part of it. The MiningFeatureOccurrence carries a geometry or location and the association with a Mining Feature provides specification of all the other descriptors. Geometry is obtained through the shape association to GM_Object.<br/>

MiningFeatureOccurrences may all be from a single map or from several maps, or reflect different levels of detail, eg point, area, 3D volume. 		Name: occurrence
-- Definition -- Tthe single MiningFeature which describes the Mining feature Occurrence
Details:
a description association that links a notional mining feature with any number of occurrence features.  A mining feature, such as a Mine may be linked to mining feature occurrences from a number of different maps or as a point, area or 3D mine plans.  A mining feature occurrence, however is always associated with only a single description (MiningFeature).<br/><br/>
